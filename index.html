About

Our Mission

Egocentric Homes is an egocentric data lab focused on modeling human hand interactions in everyday household environments. We collect, annotate, and structure first-person video with hand pose, semantic segmentation, and action labels — building foundational data for physical AI, robotics, and embodied intelligence research.

import json

# Load the Home Hands 10 annotation file
with open('assets/data/annotations.json', 'r') as f:
    data = json.load(f)

# Iterate over annotated frames
for frame_id, info in data.items():
    action    = info.get('action', 'N/A')       # Task label
    keypoints = info.get('hand_keypoints', {})  # Detected hands
    print(f"Frame {frame_id}: {action} — {len(keypoints)} hand(s)")
