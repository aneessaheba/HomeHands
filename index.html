HomeHands Dataset

Egocentric video recordings of everyday household tasks captured from a head-mounted camera. Includes RGB frames, 21-keypoint hand pose annotations per frame, and pixel-level semantic segmentation masks aligned to each recording.

Modalities

Three synchronized data streams per recording session

🎥

RGB Videos

Full-color egocentric video at 30 FPS captured at 1080p resolution.

  • Format MP4 / H.264
  • Resolution 1920 × 1080
  • Frame Rate 30 FPS
  • Duration 2 – 5 min / clip
✋

Hand Pose

Per-frame 2D hand keypoint annotations extracted via MediaPipe Hands.

  • Keypoints 21 per hand
  • Format JSON
  • Hands Left + Right
  • Confidence Score included
🎨

Segmentation

Pixel-level semantic segmentation masks aligned to every RGB frame.

  • Type Semantic
  • Format PNG masks
  • Classes 10 task classes
  • Coverage Full frame

Python 
import json

# Load the HomeHands annotation file
with open('assets/data/annotations.json', 'r') as f:
    data = json.load(f)

# Iterate over annotated frames
for frame_id, info in data.items():
    action    = info.get('action', 'N/A')       # Task label
    keypoints = info.get('hand_keypoints', {})  # Detected hands
    print(f"Frame {frame_id}: {action} — {len(keypoints)} hand(s)")
